# Node Unlock [Drupal 6.x]

Este módulo tem por objetivo desabilitar a trava imposta pelo Drupal, quando o mesmo conteúdo é editado posteriormente por outro usuário, prevalecendo a sua versão:

# Instalação:
 - Copie os arquivos do projeto para a pasta: /sites/modules, e ative-a na seção de módulos do Drupal.
